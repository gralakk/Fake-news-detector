# Fake News Detector
## Autorzy
Karolina Gralak, Wiktor Chmiel 

## Opis projektu
Projekt grupowy polegający na stworzeniu systemu klasyfikującego krótkie newsy jako prawdziwe, fałszywe lub wątpliwe z wykorzystaniem Large Language Models.

## Zakres projektu
- przygotowanie i analiza danych,
- budowa systemu klasyfikacji newsów,
- wykorzystanie modeli językowych do oceny wiarygodności treści,
- analiza wyników i prezentacja wniosków.

## Zawartość repozytorium
- `detektor_fakenews.ipynb` – notebook z implementacją projektu,
- `Raport_detektor_fakenews.pdf` – raport opisujący założenia, przebieg i wyniki projektu,
- `prezentacja_detektor_fakenews.pdf` – prezentacja podsumowująca projekt.
